# SauceDemo Automation Project

Automation testing project for https://www.saucedemo.com/

## Modules Covered
- Login
- Product Listing
- Cart
- Checkout
- Complete Flow

## Technology Used
- Python
- Playwright
- Chromium Browser

## Requirements
- Python Installed
- Playwright Installed
- Chromium Browser

## Install Playwright

```bash
pip install playwright
```

## Install Browser

```bash
playwright install
```

## Run Test Scripts

```bash
python Loginsaucedemo.py

python productlistsaucedemo.py

python cartsaucedemo.py

python checkoutsaucedemo.py

python completeflowsaucedemo.py
```
